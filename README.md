# AHK Layer Switcher
## 概要

AutoHotkeyを使用して自作キーボードのようなレイヤー機能を実現します。

具体的には以下の2つのレイヤーを追加します。

* ショートカットレイヤー
  * 様々なショートカットやユーティリティ系マクロを組み込んだ便利なレイヤーです。
* マウスカーソルレイヤー
  * マウスによる操作を代替します。

## 必要条件

* AutoHotKey 1.1.34.04

## 使用方法&インストール

Autohotkeyで `./layerChengeMain.ahk` を実行してください。
